# Pokedex Clone

A simple **Pokedex** app built with React. It fetches data from **PokeAPI** and provides search, type filtering, pagination, and a Pokémon details page.

## Live Demo
pokedex-tau-silk.vercel.app

## Tech Stack
- React (Vite)
- React Router
- Redux Toolkit + RTK Query
- Material UI (MUI)
- Tailwind CSS
- PokeAPI

## Features
- Pokémon list (up to 1350)
- Search by name
- Filter by type
- Pagination (next/prev + counter)
- Details page (stats, sprites, basic info, cry audio)

## Run Locally
```bash
npm install
npm run dev
